## Features

- **Window Manager**: Hyprland
- **Bar**: Waybar
- **Terminal**: Kitty
- **Shell**: Zsh
- **Launcher**: Rofi
- **File Manager**: Yazi
- **Wallpaper**: hyprpaper
- **Lock Screen**: hyprlock
- **Idle Management:** hypridle
- **Cursor:** hyprcursor
- **Discord Client**: Vesktop

## Recommended packages
- **Fonts**: `noto-fonts` and `otf-font-awesome` (for icons)
